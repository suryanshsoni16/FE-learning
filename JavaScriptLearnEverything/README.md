# 45 Days Frontend Interview Preparation Kit

# How To Use This Roadmap

## If Starting From Zero Again

### Daily Plan

**Weekdays (3–4 hrs)**

- 1.5 hr Learn concepts
- 1 hr Coding / Practice
- 1 hr Revision

**Weekend (6 hrs)**

- Build projects
- Mock interviews
- Revise weak areas
